- 🔄 **Классические фигуры Лиссажу** с выбором соотношений
- 🧊 **3D-фигуры** с вращением
- 🎚️ **Калибровка**: Gain X/Y, Offset, Фаза
- 🎵 **Экспорт в стерео WAV** (X = левый канал, Y = правый)
- ♾️ **Бесконечный цикл** для реального времени на осциллографе/лазере

---

## 🛠 Технологии

- **Чистый стек**: HTML + CSS + JavaScript (без внешних библиотек)
- **Генерация сигнала**: Canvas → математические кривые → WAV
- **Экспорт**: 16-bit стерео WAV файл
- **Адаптивность**: отлично работает на ПК и мобильных устройствах

---

## 📲 Как использовать

1. Откройте **[демо](https://volkovhl.github.io/XY-Figure-Generator/)**
2. Выберите вкладку (Фигуры / Текст / Лиссажу / 3D)
3. Настройте параметры (фигура, частота, фаза, gain/offset)
4. Нажмите **СГЕНЕРИРОВАТЬ WAV**
5. Скачайте файл и воспроизведите на осциллографе или лазерном проекторе

> 💡 **Совет**: Для лазерных шоу используйте низкие частоты (0.5–2 Гц) и хорошую калибровку по фазе.

---

## 🧠 Что внутри

### Режимы генерации
- **Фигуры** — базовые геометрические примитивы
- **Текст** — векторизация введённого текста
- **Лиссажу** — классические фигуры с настраиваемым соотношением частот
- **3D** — трёхмерные объекты с вращением

### Калибровка
- Фаза (0–360°)
- Gain X / Gain Y
- Offset X / Offset Y
- Длительность сигнала
